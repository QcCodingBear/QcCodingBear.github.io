Programmation PL/SQL

Introduction

PL/SQL est un langage procédural intégré dans SQL, permettant de combiner des instructions SQL avec une logique procédurale.

Structure d'un bloc PL/SQL

Un bloc PL/SQL est divisé en trois sections :

  • DECLARE : Déclare les variables, constantes et curseurs.
  • BEGIN : Contient les instructions SQL et PL/SQL.
  • EXCEPTION : Gère les exceptions (erreurs).

Exemple :

DECLARE
  v_nom CHAR(15) := 'JULES';
BEGIN
  DBMS_OUTPUT.PUT_LINE('Bonjour, ' || v_nom);
EXCEPTION
  WHEN OTHERS THEN
    DBMS_OUTPUT.PUT_LINE('Erreur détectée');
END;

Les Variables

Les variables sont déclarées dans la section DECLARE. Voici quelques types et syntaxes :

  • Type simple : v_salaire NUMBER(7,2);
  • Basé sur une colonne : v_nomE emp.NomE%TYPE;
  • Basé sur une ligne : v_record emp%ROWTYPE;

Traitements Conditionnels

Structure IF...THEN...ELSIF

IF condition1 THEN
  -- Instructions
ELSIF condition2 THEN
  -- Instructions
ELSE
  -- Instructions
END IF;

Structure CASE

CASE expression
  WHEN valeur1 THEN
    -- Instructions
  WHEN valeur2 THEN
    -- Instructions
ELSE
    -- Instructions
END CASE;

Traitements Répétitifs

La Boucle WHILE

WHILE condition LOOP
  -- Instructions
END LOOP;

La Boucle FOR

Permet d'itérer sur une plage de valeurs :

FOR i IN 1..10 LOOP
  DBMS_OUTPUT.PUT_LINE('Valeur de i: ' || i);
END LOOP;
