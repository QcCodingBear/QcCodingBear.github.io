Les Packages SQL

Qu'est-ce que c'est ?

Un package SQL est un regroupement d'éléments de programmation PL/SQL, comme des procédures, des fonctions, des variables, et plus encore. Il agit comme un conteneur permettant d'organiser et de réutiliser ces éléments.

Structure d'un Package

Un package est divisé en deux parties :

  • Spécification (spec) : Déclare les éléments publics accessibles de l'extérieur.
    CREATE OR REPLACE PACKAGE my_package AS
    PROCEDURE add_employee(emp_id NUMBER, emp_name VARCHAR2);
    FUNCTION get_employee_count RETURN NUMBER;
    END my_package;
  • Corps (body) : Contient les définitions des éléments déclarés dans la spécification.
    CREATE OR REPLACE PACKAGE BODY my_package AS
    PROCEDURE add_employee(emp_id NUMBER, emp_name VARCHAR2) IS
    BEGIN
    -- Logic for adding an employee
    DBMS_OUTPUT.PUT_LINE('Employee added: ' || emp_name);
    END add_employee;

    FUNCTION get_employee_count RETURN NUMBER IS
    emp_count NUMBER;
    BEGIN
    -- Logic to count employees
    emp_count := 100; -- Placeholder value
    RETURN emp_count;
    END get_employee_count;
    END my_package;

À quoi ça sert ?

Les packages servent à :

  • Organiser le code par fonctionnalités liées.
  • Réutiliser des éléments dans différents contextes.
  • Encapsuler la logique interne et limiter l'accès à certains éléments.
  • Améliorer les performances en chargeant efficacement le package en mémoire.

Exemple de Package

Spécification :

Imaginons un package pour gérer un système d'inventaire

CREATE OR REPLACE PACKAGE inventory_pkg AS
PROCEDURE update_stock(product_id NUMBER, quantity NUMBER);
FUNCTION get_stock(product_id NUMBER) RETURN NUMBER;
END inventory_pkg;

Corps :

CREATE OR REPLACE PACKAGE BODY inventory_pkg AS
PROCEDURE update_stock(product_id NUMBER, quantity NUMBER) IS
BEGIN
DBMS_OUTPUT.PUT_LINE('Stock mis à jour pour le produit : ' || product_id);
END update_stock;

FUNCTION get_stock(product_id NUMBER) RETURN NUMBER IS
stock NUMBER;
BEGIN
stock := 50; -- Valeur fictive
RETURN stock;
END get_stock;
END inventory_pkg;

Utilisation :

BEGIN
inventory_pkg.update_stock(101, 20);
DBMS_OUTPUT.PUT_LINE('Stock actuel : ' || inventory_pkg.get_stock(101));
END;
